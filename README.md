# 📻 Gerador POP FM v3.1 - OS_SYSTEM_AUTOMATION

Central de Controle de Emissão Profissional com Dashboard Analítico e Motor de Seleção Inteligente.

## 🚀 Instalação Rápida

Para rodar este projeto em sua máquina local:

1. **Requisitos:**
   - Python 3.10 ou superior
   - Node.js (v16+)

2. **Clone o repositório:**
   ```bash
   git clone https://github.com/NettoNunes10/gerador-pop-local.git
   cd gerador-pop-local
   ```

3. **Configure o ambiente:**
   ```bash
   python setup_local.py
   ```

4. **Inicie o sistema:**
   ```bash
   python run.py
   ```

## 💎 Recursos v3.1
- **Dashboard Analítico:** Gráficos interativos (Recharts) de equilíbrio de categorias e artistas.
- **Player Integrado:** Pré-escuta de qualquer música da biblioteca diretamente no navegador.
- **Motor de Scoring:** Seleção baseada em tempo de descanso, BPM e multiplicadores de artistas favoritos.
- **Wildcards (Surpresas):** Sistema de substituição aleatória de categorias (ex: B vira C) configurável via UI.
- **Mapeamento Semanal:** Selecione um modelo `.blm` diferente para cada dia da semana.

## 🛠️ Manutenção e Atualização
Sempre que o desenvolvedor aplicar uma nova funcionalidade, basta rodar o comando abaixo na sua pasta local para baixar as atualizações:

```bash
git pull origin main
```
